# request-bucket

## Configuration

- `OPENSEARCH_ENDPOINT`
  - Specify OpenSearch endpoint
  - Example: `https://opensearch.example.com/`
- `OPENSEARCH_USERNAME`
  - Specify OpenSearch user name
- `OPENSEARCH_PASSWORD`
  - Specify OpenSearch password
- `OPENSEARCH_INDEX`
  - Specify OpenSeach index to store bucket data
